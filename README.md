# 🛒 Django E-Commerce Platform (Full-Featured)

This project is a modern web-based e-commerce application that covers
the complete shopping workflow --- from user registration to order
management.

## 🌟 Key Features

-   **User System (Auth):** User registration, login, logout, and
    profile management.
-   **Dynamic Cart:** Session-based shopping cart with real-time
    quantity updates.
-   **Order & Payment Simulation:** Collects shipping address
    information and stores order history.
-   **Product Management:** Category-based filtering, product search
    functionality, and product images.
-   **Advanced Admin Control:** Track orders and manage products through
    the Django admin panel.

## 🛠️ Technical Stack

-   **Backend:** Python 3.x, Django Framework
-   **Frontend:** HTML5, CSS3, Bootstrap 5, Bootstrap Icons
-   **Authentication:** Django Built-in User Authentication System
-   **Database:** SQLite (Relational Database)

## 🎯 Project Purpose

This project was developed as a comprehensive learning exercise to
master Django's MVT architecture, session management, and database
relations.
